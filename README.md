# Las Vegas Mahjong

Website for **Las Vegas Mahjong** — mahjong lessons, open play events, leagues, and tournaments across Las Vegas, Summerlin, and Henderson.

**Live site:** [lasvegasmahj.com](https://lasvegasmahj.com)

## About

Las Vegas Mahjong is run by Shauna, a certified Oh My Mahjong (OMM) instructor. The business offers:

- Beginner, intermediate, and advanced mahjong lessons
- Open play events across the Las Vegas Valley
- Leagues and tournaments
- Corporate and private events

## Sister Site

[Find My Mahj Game](https://findmymahjgame.com) — a national directory to find mahjong players, groups, and events near you. ([GitHub repo](https://github.com/lasvegasmahj-maker/findmymahjgame))

## Tech Stack

- Static HTML/CSS/JavaScript (single-page site)
- Hosted on GoDaddy
- Mailchimp for newsletter
- No build step required

## Preview Locally

open index.html
